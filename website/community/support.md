Se você precisa de ajuda com seu aplicativo React Native, o lugar certo para ir depende do tipo de ajuda que você precisa.

### Repository

O núcleo do **React Native** é trabalhado em tempo integral pela equipe React Native da Meta. Mas há muito mais pessoas na comunidade que fazem contribuições importantes e corrigem coisas. Se o problema que você está enfrentando é relacionado a código, você deve verificar os issues abertos no [repositório principal](https://github.com/facebook/react-native/issues).

Se você não conseguir encontrar um issue existente, por favor consulte [Como reportar um bug](/contributing/how-to-report-a-bug).

### Suporte para upgrade

Muitas vezes, ao fazer upgrade de seus aplicativos e bibliotecas de uma versão do React Native para uma mais recente, você pode precisar de alguma ajuda; a comunidade se uniu para criar alguns recursos importantes:

- [upgrade-helper](https://react-native-community.github.io/upgrade-helper/) é uma ferramenta que mostrará o conjunto completo de mudanças acontecendo entre quaisquer duas versões para ajudá-lo a ver quais mudanças você precisa fazer no seu código.
- [upgrade support](https://github.com/react-native-community/upgrade-support) é um repositório impulsionado pela comunidade para solicitar e oferecer ajuda ao fazer upgrade do seu aplicativo.

### Stack Overflow

Muitos membros da comunidade usam Stack Overflow para fazer perguntas. Leia as [perguntas existentes](https://stackoverflow.com/questions/tagged/react-native?sort=frequent) marcadas com **react-native** ou [faça a sua própria](https://stackoverflow.com/questions/ask?tags=react-native)!

### Reactiflux Chat

Se você precisa de uma resposta imediatamente, confira a comunidade [Reactiflux Discord](https://discord.gg/JuTwWB8rsy). Geralmente há vários especialistas em React Native lá que podem ajudar ou apontá-lo para algum lugar que você possa querer procurar.
