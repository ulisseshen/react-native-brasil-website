Recentemente compartilhamos [a Visão de Muitas Plataformas do React Native](https://reactnative.dev/blog/2021/08/26/many-platform-vision) sobre como expandir o React para outras plataformas melhora o framework para todos os outros. Temos feito progresso significativo nesta visão nos últimos dois anos ao fazer parceria com a Microsoft no React Native para Windows e macOS, e com o Oculus no React Native em VR.

Como [parte de nossos planos iniciados no início deste ano](https://reactnative.dev/blog/2021/08/19/h2-2021), estamos expandindo nosso foco nessas plataformas e expandindo nossas equipes para nos ajudar a alcançar nossa visão. Para apoiar nossos novos colegas de equipe, e muitos mais por vir, **estamos contratando dois Gerentes de Engenharia: um para ajudar a apoiar o React Native para Desktop, e um para apoiar o React Native para VR**.

Estamos procurando gerentes que se importem profundamente com o crescimento e sucesso de nossos engenheiros, e estejam animados com nossa visão. Experiência prévia em React ou React Native não é necessária. Esta função está aberta para qualquer pessoa nos Estados Unidos. Se isso parece uma oportunidade interessante e você está interessado em saber mais, por favor candidate-se na [Página de Carreiras do Facebook](https://www.facebook.com/careers/v2/jobs/438516437547870). Aguardamos ouvir de você!
