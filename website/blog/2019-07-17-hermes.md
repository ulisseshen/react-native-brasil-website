Na semana passada na Chain React, anunciamos o Hermes, um motor JavaScript de código aberto no qual estamos trabalhando no Facebook. É um motor JavaScript pequeno e leve otimizado para executar React Native no Android. [Confira!](https://code.fb.com/android/hermes/)

O Hermes melhora o desempenho do React Native ao diminuir a utilização de memória, reduzir o tamanho do download e diminuir o tempo que leva para o app se tornar utilizável ou "tempo até interatividade" (TTI).

> "Ao analisarmos os dados de desempenho, notamos que o motor JavaScript em si era um fator significativo no desempenho de inicialização e tamanho do download. Com esses dados em mãos, sabíamos que precisávamos otimizar o desempenho do JavaScript nos ambientes mais restritos de um celular em comparação com um desktop ou laptop. Depois de explorar outras opções, construímos um novo motor JavaScript que chamamos de Hermes. Ele é projetado para melhorar o desempenho do app, com foco em nossos apps React Native, mesmo em dispositivos de mercado de massa com memória limitada, armazenamento lento e poder de computação reduzido." —[Hermes: An open source JavaScript engine optimized for mobile apps, starting with React Native](https://code.fb.com/android/hermes/)

Quer começar imediatamente? Não deixe de [conferir nosso novo guia para habilitar o Hermes em seu app React Native existente](/docs/hermes/) na documentação!
