Hoje estamos lançando a versão 0.65 do React Native com uma nova versão do Hermes, melhorias de acessibilidade, atualizações de pacotes e muito mais.

## O que há de novo no Hermes 0.8?

[Hermes](https://hermesengine.dev), a VM JavaScript open source do Facebook otimizada para React Native, foi atualizada para a versão 0.8.1. Alguns dos recursos de destaque neste lançamento são:

- Um novo coletor de lixo concorrente intitulado "Hades" que oferece até 30 vezes tempos de pausa mais curtos em dispositivos de 64 bits. No Facebook, vimos isso melhorar algumas cargas de trabalho intensivas em CPU em 20%-50%. Você pode [saber mais sobre o Hades aqui](https://hermesengine.dev/docs/hades/).
- [API de Internacionalização ECMAScript (ECMA-402, ou `Intl`)](https://hermesengine.dev/docs/intl) agora está integrada ao Hermes no Android e habilitada por padrão, com apenas 57-62K por sobrecarga de tamanho da API (comparado aos [6MiB do JSC](https://github.com/react-native-community/jsc-android-buildscripts)). Com esta mudança, os usuários do Hermes não precisam mais de polyfills de localização. Um grande agradecimento a [@mganandraj](https://github.com/mganandraj) e outros parceiros da Microsoft por impulsionar a implementação para tornar isso possível!
- [Hermes no iOS](/blog/2021/03/12/version-0.64) agora [suporta Apple M1 Macs e Mac Catalyst](https://github.com/facebook/hermes/pull/546)!
- Melhorias de memória incluindo SMI (Small Integers) e compressão de ponteiro que [reduziram o heap JS em 30%](https://twitter.com/tmikov/status/1385629737121243140).
- Mudanças no `Function.prototype.toString` que [corrigiram uma queda de desempenho devido à detecção inadequada de recursos](https://github.com/facebook/hermes/issues/471#issuecomment-820123463) e [suportam o caso de uso de injeção de código-fonte](https://github.com/facebook/hermes/issues/114).

Você pode encontrar o [changelog completo do Hermes aqui](https://github.com/facebook/hermes/releases).

[Siga os passos aqui](/docs/hermes#enabling-hermes) para habilitar o Hermes em seu aplicativo, se ainda não o fez, para aproveitar esses novos recursos e ganhos!

## Correções e Adições de Acessibilidade

No ano passado [o Facebook aceitou o compromisso GAAD](https://reactnative.dev/blog/2021/05/20/GAAD-One-Year-Later) para melhorar a acessibilidade no React Native. A versão 0.65 compartilha os resultados deste compromisso e outras vitórias de acessibilidade! Algumas mudanças notáveis incluem:

- Permitir especificação de valores de alto contraste claro e escuro para iOS. Veja a [documentação](/docs/dynamiccolorios) para mais detalhes.
- Adicionada API [`getRecommendedTimeoutMillis`](/docs/accessibilityinfo#getrecommendedtimeoutmillis-android) no Android. Isso expõe o valor de timeout padrão preferido do usuário conforme definido nas opções de acessibilidade do Android e é para usuários que podem precisar de tempo extra para revisar ou alcançar controles, etc.
- Correções gerais para garantir que o TalkBack/VoiceOver anuncie adequadamente os estados da UI como `disabled` e `unselected` nos componentes.

Você pode acompanhar ou contribuir para nossas [issues de acessibilidade pendentes](https://github.com/facebook/react-native/projects/15) aqui!

## Atualizações Notáveis de Versão de Dependências e Observações

- `react-native-codegen` versão `0.0.7` agora é necessário como `devDependency` no `package.json`.
- O JCenter foi descontinuado e agora é somente leitura. Removemos o JCenter como repositório maven e atualizamos as dependências para usar MavenCentral e Jitpack.
- Atualizado OkHttp da v3 para v4.9.1. Veja [Atualizando para OkHttp 4](https://square.github.io/okhttp/upgrading_to_okhttp_4/) para mais detalhes sobre as mudanças.
- Atualizado para Flipper 0.93 para suportar Xcode 12.5. Veja [changelog do Flipper aqui](https://github.com/facebook/flipper/blob/master/desktop/static/CHANGELOG.md).
- Suporte para Android Gradle Plugin 7
- Apple Silicon requer um workaround de linker. Veja [nota de @mikehardy](https://github.com/react-native-community/releases/issues/238#issuecomment-890367992) sobre isso.
