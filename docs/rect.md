---

`Rect` aceita valores numéricos em pixels para descrever o quanto deve-se estender uma área retangular. Esses valores são adicionados ao tamanho da área original para expandi-la.

## Exemplo

```js
{
    bottom: 20,
    left: null,
    right: undefined,
    top: 50
}
```

## Chaves e valores

### `bottom`

| Tipo                        | Obrigatório |
| --------------------------- | ----------- |
| number, `null`, `undefined` | Não         |

### `left`

| Tipo                        | Obrigatório |
| --------------------------- | ----------- |
| number, `null`, `undefined` | Não         |

### `right`

| Tipo                        | Obrigatório |
| --------------------------- | ----------- |
| number, `null`, `undefined` | Não         |

### `top`

| Tipo                        | Obrigatório |
| --------------------------- | ----------- |
| number, `null`, `undefined` | Não         |
